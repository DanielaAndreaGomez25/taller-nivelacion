 Fundamentos de Programación en Java

15. Tipos de datos primitivos

- int
- double
- float
- char
- boolean
- byte
- short
- long

 16. Estructuras de control

- if / else: permiten tomar decisiones.
- switch: evalúa múltiples opciones.
- for: bucle con contador.
- while: repite mientras se cumpla una condición.
- do-while: se ejecuta al menos una vez.

17. Importancia de nombres significativos

Permiten que el código sea claro y fácil de entender.

 18. ¿Qué es la Programación Orientada a Objetos?

Es un paradigma basado en clases y objetos que contienen atributos y métodos.

 19. Cuatro pilares de la POO

- Encapsulamiento
- Abstracción
- Herencia
- Polimorfismo

 20. ¿Qué es la herencia?

Permite que una clase herede atributos y métodos de otra usando extends.

 21. Modificadores de acceso

- public
- private
- protected
- default

 22. ¿Qué es una variable de entorno?

Es una variable del sistema que almacena configuraciones necesarias para ejecutar programas, como JAVA_HOME.
